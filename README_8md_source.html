4 YarrPP is a C++ library that a pirate would want to have.
5 It is a random collection of files that was used to write the articles on
6 [C++ Pirate](http://www.gockelhut.com/cpp-pirate/index.html), which is where I write things about C++ when the mood
7 strikes me.
8 Is it production-grade?
9 Probably not.
10 But it is [documented](http://tgockel.github.io/yarrpp) and [tested](https://travis-ci.org/tgockel/yarrpp).
